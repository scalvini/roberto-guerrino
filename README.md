Sito commemorativo in due lingue, stile tipografico svizzero (Helvetica, bianco e nero).

## File

- `index.html`  pagina italiana (predefinita)
- `en.html`  pagina inglese
- Le due pagine sono collegate tra loro dal selettore di lingua in alto a destra.

## Pubblicare su GitHub Pages (circa due minuti)

1. Vai su github.com e crea un nuovo repository, per esempio `roberto-guerrino`. Lascialo pubblico.
2. Nella pagina del repository scegli "Add file" e poi "Upload files".
3. Trascina `index.html` e `en.html` (e questo `README.md` se vuoi), poi conferma con "Commit changes".
4. Apri "Settings", quindi la voce "Pages" nel menu a sinistra.
5. In "Source" seleziona il branch `main` e la cartella `/ (root)`, poi salva.
6. Dopo circa un minuto il sito sarà online a un indirizzo del tipo `https://TUO-UTENTE.github.io/roberto-guerrino/`. La pagina inglese sarà a `.../en.html`.
